# VNC Tools

VNC-related utilities for remote GUI access.

## Files
- `vnc_web_viewer.py` - Web-based VNC viewer
- `start_gui_with_vnc.sh` - Script to launch GUI with VNC support

## Usage
These tools are for advanced remote desktop access scenarios.

### Start GUI with VNC
```bash
cd vnc_tools
./start_gui_with_vnc.sh
```

### Web Viewer
```bash
cd vnc_tools
python vnc_web_viewer.py
```
