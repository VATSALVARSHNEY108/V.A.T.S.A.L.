# VATSAL Chatbot

Intelligent chatbot with CLI interface using Google Gemini AI.

## Features
- Natural language conversations
- Context-aware responses
- Conversation history tracking
- Statistics and analytics
- Session management

## How to Run
```bash
cd vatsal_chatbot
python vatsal_chatbot.py
```

## Requirements
- GEMINI_API_KEY environment variable
- vatsal_ai module
- Google Gemini AI access
