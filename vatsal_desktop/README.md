# VATSAL Desktop Automator

Intelligent desktop automation with local execution and AI understanding.

## Features
- Desktop automation (open apps, files, folders)
- System control (lock, shutdown, restart)
- File management
- Screenshot capabilities
- Mouse and keyboard control
- System monitoring
- Process management

## How to Run
```bash
cd vatsal_desktop
python vatsal_desktop_automator.py
```

## Requirements
- GEMINI_API_KEY environment variable
- PyAutoGUI for GUI automation
- psutil for system monitoring
- Google Gemini AI access
